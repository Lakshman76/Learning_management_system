# Learning Management System
## Frontend part
### Setup Instructions

1. clone the project 

``` 
    git clone https://github.com/Lakshman76/Learning_management_system/tree/main/client
```

2. Move into the directory

``` 
    cd Learning_management_system
```

3. Install dependencies

```
    npm install
```

4. Run the server

``` 
    npm run dev
```
